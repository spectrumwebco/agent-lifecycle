
## Install Virtualbox

### Recommended machine requirements

Based on [Virtualbox](https://www.virtualbox.org/wiki/End-user_documentation) and 
[Ubuntu Desktop](https://ubuntu.com/download/desktop) documentation the following are the
minimum machine requirement:

| Item | Minimum requirement|
|---|---|
| CPU | dual core CPU |
| RAM | 8GB |
| Storage | 30GB |
| Network Connectivity | access to public internet |

### Virtualbox Installer

Download and install [Virtualbox](https://www.virtualbox.org/wiki/Downloads) on your machine.
